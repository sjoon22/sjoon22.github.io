## Overview

This project examines how multi-modal systems behave when text and image signals are partially conflicting.
The goal is to identify practical indicators of uncertainty for downstream decision support.

## What I did

- Built a small benchmark set with controlled text-image mismatch cases
- Compared model outputs across prompt templates
- Analyzed consistency patterns and potential confidence signals
