## Overview

This project builds a reproducible evaluation setup for LLM safety behavior.
It supports scenario-based prompt sets, rule-based checks, and manual review logs.

## What I did

- Designed prompt scenarios for risky and ambiguous user intents
- Implemented a structured evaluation template for response quality and safety
- Documented known failure modes and mitigation ideas
